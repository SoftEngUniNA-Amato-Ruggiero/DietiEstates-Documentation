\section{Tecnologie utilizzate}
Il frontend DietiEstates25 è sviluppato come Single Page Application (SPA) utilizzando tecnologie web moderne che garantiscono un'esperienza utente fluida.

\subsection{Angular 20}
Angular è un framework completo per la costruzione di applicazioni web moderne, in particolare di Single Page Application (SPA). A differenza di altre soluzioni, Angular è più opinionated, e fornisce fin dall'inizio un sistema integrato che copre la maggior parte delle necessità di un'applicazione complessa.\\
Angular permette di organizzare l'interfaccia in componenti, unità modulari, quindi riutilizzabili, che incapsulano logica, template HTML e stili CSS. Questa struttura favorisce la separazione delle responsabilità e rende l'applicazione più manutenibile e scalabile. Un altro elemento centrale è il sistema di Dependency Injection; è il framework a gestire le dipendenze tra i vari servizi e componenti, riducendo l'accoppiamento tra le parti dell'applicazione.\\
Angular include un modulo di routing nativo, ossia permette di gestire la navigazione tra diverse viste dell'applicazione senza dover ricaricare la pagina, simulando il funzionamento di un sito multipagina in una SPA. Per la gestione dell'input utente, il framework mette a disposizione le Reactive Forms, un approccio fortemente dichiarativo e tipizzato che facilita la creazione di form complessi con annessa validazione finale dei dati, sia sincrona che asincrona.\\
Dal punto di vista della comunicazione con il backend, Angular fornisce un HTTP Client integrato pensato per interagire con API REST e servizi esterni.\\
Nel complesso Angular si propone come soluzione particolarmente adatta allo sviluppo di applicazioni medio-grandi.

\subsection{Typescript}
Typescript è un superset di JavaScript che aggiunge tipizzazione statica al linguaggio. Ogni tipo è esplicito a tempo di compilazione.\\
Le inconsistenze sono segnalate dal compilatore, accelerando di gran lunga lo sviluppo dell'applicazione. Inoltre i tipi fungono da documentazione stessa essendo autoesplicativi.

\subsection{Bootstrap}
Bootstrap è un framework CSS che fornisce componenti UI predefinite con una griglia responsiva di supporto. Per componenti più sofisticati è stato utilizzato Angular Material, cercando di utilizzare il meglio di entrambi i framework.\\

\textbf{Caratteristiche principali}
\begin{itemize}
    \item \textbf{Responsive Design}: Il layout si adatta a schermi di diverse dimensioni (dektop, mobile), fondamentale per una qualsiasi applicazione web moderna.
    \item \textbf{Componenti pronti}: Navbar, modal, card, etc. Tutti con stili consistenti.
    \item \textbf{Riduzione CSS}: Non è necessario scrivere codice CSS per gli elementi in comune.
\end{itemize}

\subsection{Geoapify}
Geoapify è un provider esterno che fornisce servizi di geocoding e mappe.
